# The Injured Pilot
## Welcome to my first 2D game!!

The structure of the game states of my project:

1. **__Intro__**
2. **__Hanger__**
  - states from A1 to A11
  - states from B1 to B6
3. **__Game Over__**
  - Bad end
  - two good endings,S1 and S2
